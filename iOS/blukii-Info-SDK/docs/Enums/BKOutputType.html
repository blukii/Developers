BKOutputType

public enum BKOutputType : String

Definition of available output types that can be chosen for a blukii number on blukii Info Manager (https://manager.blukiiinfo.com)

It will be returned by BKInfoOutputData.outputType

  • PDF

    According to the BKOutputType the BKResolveData is affected:

    • BKInfoOutputData.url returns a url of an PDF-File;
    • BKInfoOutputData.urlExtra returns a url of an audio file
    • BKInfoOutputData.plainTextExtra returns a translated text
