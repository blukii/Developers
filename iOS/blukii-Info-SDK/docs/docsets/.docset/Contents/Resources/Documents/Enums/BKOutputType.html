BKOutputType

public enum BKOutputType: String

Definition of available output types that can be chosen for a blukii number on blukii Info Manager (https://manager.blukiiinfo.com)

It will be returned by BKResolveData.outputType

  • PDF

    According to the BKOutputType the BKResolveData is affected:

    • BKResolveData.url returns a url of an PDF-File;
    • BKResolveData.urlExtra returns a url of an audio file
    • BKResolveData.plainTextExtra returns a translated text

    Declaration

    Swift

    case PDF = "PDF"
  • YOUTUBE

    According to the BKOutputType the BKResolveData is affected:

    BKResolveData.url returns a url of a youtube video

    Declaration

    Swift

    case YOUTUBE = "YOUTUBE"
  • VCARD

    According to the BKOutputType the BKResolveData is affected:
