BKClientNewsDelegate

public protocol BKClientNewsDelegate

Delegate for retreiving a translated blukii Info Manager news text.

The Delegate are registered by BKClient.newsDelegate

  • onNewsResult(_:)

    Will be called once after calling BKClient.resolveServerNews(languageId: String)

    If the element can not be resolved because of no data or any error, onNewsError(errorCode: BKClient.ErrorCode) will be called instead.
