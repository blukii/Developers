BKDecryptSecureBeaconsDelegate

public protocol BKDecryptSecureBeaconsDelegate

Delegate for the SecureBeacon decryption

Decrypt Delegate is registred by BKClient.decryptSecureBeaconDelegate

  • onDecryptSecureBeacons(_:decryptedCount:)

    Will be called if BKClient.decryptSecureBeacons(discoveryDataList:) is successfully requested. Parameter decryptedDiscoveryDataList contains all list items contained as parameter in BKClient.decryptSecureBeacons(discoveryDataList:). If the authenticated blukii Manager user is owner of list items that are marked with BKDiscoveryData.secureBeaconState as BKSecureBeaconState.encrypted, these are decrypted.
