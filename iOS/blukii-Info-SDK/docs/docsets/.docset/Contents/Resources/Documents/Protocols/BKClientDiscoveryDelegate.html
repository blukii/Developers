Delegate for the BLE discovery.

Discovery delegates are registred by BKClient.discoveryDelegate

  • onDiscoveryResult(_:)

    Will be called frequently after one scan duration for returning the scan results.

    If no BLE modules has been found discoveryResult list is an empty list (not nil).

    If BKResolveSettings.resolceEnabled is true but the blukii Info Manager is not accessible this delegate will be called anyway with a discovery result BKOutputElement list but without BKResolveData objects. In this case onDiscoveryError(errorCode: BKClient.ErrorCode) will be called additionally

    If the discovery failed because of an error onDiscoveryResult(discoveryResultList: [BKOutputElement]) will not be called.
